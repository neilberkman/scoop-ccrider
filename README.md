# Scoop bucket for ccrider

[Scoop](https://scoop.sh) bucket for [ccrider](https://github.com/neilberkman/ccrider) - Search, browse, and resume your Claude Code sessions.

## Installation

```bash
scoop bucket add ccrider https://github.com/neilberkman/scoop-ccrider
scoop install ccrider
```

## About ccrider

ccrider is a CLI tool for searching and browsing Claude Code session history. It provides full-text search with SQLite FTS5, an interactive TUI, and smart session resumption with context.

See the [main repository](https://github.com/neilberkman/ccrider) for more information.
